=== Base Conformance Class

The Base Conformance Class implements the following Requirements Classes from the CityGML 3.0 Conceptual Model Standard:

* Core: http://www.opengis.net/spec/CityGML-1/3.0/req/req-class-core
* ADE: http://www.opengis.net/spec/CityGML-1/3.0/req/req-class-ade

The applicable GML schema is http://schemas.opengis.net/citygml/3.0/cityGMLBase.xsd[cityGMLBase.xsd]
